Hello! Welcome to my newly upgraded portfolio. I've transitioned from a single-file HTML site to a powerful **Jekyll** architecture.

### Why Jekyll?

Jekyll allows me to:
- Manage blog posts easily with **Markdown**.
- Keep my code DRY (Don't Repeat Yourself) using layouts and includes.
- Leverage GitHub Pages' native support for static site generation.

### What's New?

1. **Interactive 3D Background**: Powered by Three.js.
2. **Multiple Themes**: Try the Cyberpunk or Vintage Manga modes!
3. **Dedicated Blog**: You're reading it right now.
4. **Project Filtering**: Coming soon to the projects page.
